## 🚀 接下来你需要做什么？

### 第一步：部署智能合约（5分钟）

**推荐使用 Remix（最简单）**：

1. 打开 https://remix.ethereum.org/
2. 创建新文件 `RedPacket.sol`
3. 复制 `RedPacket.sol` 的内容并粘贴
4. 编译合约（选择 Solidity 0.8.0+）
5. 部署到 Monad 网络：
   - Environment: Injected Provider - MetaMask
   - 确保 MetaMask 连接到 Monad（Chain ID: 143）
   - 点击 Deploy
6. **复制部署的合约地址**（重要！）

### 第二步：配置前端（1分钟）

1. 打开 `app.html` 文件
2. 找到第 46 行：
   ```javascript
   const CONTRACT_ADDRESS = '0x0000000000000000000000000000000000000000';
   ```
3. 将 `0x0000...` 替换为你刚才复制的合约地址
4. 保存文件

### 第三步：运行应用（1分钟）

**方法 1：使用 Python**（如果已安装）
```bash
cd /workspace
python3 -m http.server 8000
```
然后访问：http://localhost:8000/app.html

**方法 2：使用 Node.js**（如果已安装）
```bash
cd /workspace
npx serve .
```
然后访问显示的 URL

**方法 3：直接打开**
- 直接双击 `app.html` 文件
- 在浏览器中打开

### 第四步：开始使用（2分钟）

1. 点击"连接钱包"
2. 在 MetaMask 中确认连接
3. 如果提示添加网络，点击"添加"：
   - Network Name: Monad Testnet
   - RPC URL: https://rpc3.monad.xyz
   - Chain ID: 143
   - Currency Symbol: MON
4. 发送你的第一个红包！🎉

## 💡 重要提示

### ⚠️ 安全警告

1. **私钥安全**：
   - 永远不要分享你的私钥
   - 不要将私钥提交到 Git
   - 使用环境变量管理敏感信息

2. **测试网 vs 主网**：
   - 先在测试网充分测试
   - 测试网代币没有实际价值
   - 主网部署前进行安全审计

3. **合约地址**：
   - 确保配置正确的合约地址
   - 不要连接到未知合约
   - 验证合约代码

### 💰 费用说明

- **部署合约**：需要 MON 支付 Gas 费（约 0.02-0.05 MON）
- **发送红包**：需要红包金额 + Gas 费
- **领取红包**：需要 Gas 费（约 0.001-0.003 MON）
- **查看历史**：免费（读取操作）
